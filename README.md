# 🔒 Password Generator — Day 7 of #10DaysOfWebDev

A simple password generator that creates strong, random passwords instantly. Choose your length, toggle numbers and symbols, and copy with one click!


## ✨ Features

- Generate random passwords instantly
- Choose password length (4 to 30 characters)
- Toggle numbers and symbols on or off
- Copy password to clipboard with one click
- Password strength checker (Weak / Medium / Strong)
- Clean dark themed UI

---

## 🛠️ Built With

- HTML
- CSS
- JavaScript (Vanilla)

---

## 📁 Project Structure

```
password-generator/
├── index.html
├── style.css
└── script.js
```

---

## 🚀 How to Run

1. Download or clone the files
2. Open `index.html` in your browser
3. Select your options and click Generate Password!

---

## 💡 What I Learned

- Generating random characters using `Math.random()`
- Reading checkbox and number input values in JavaScript
- Using `navigator.clipboard.writeText()` to copy text
- Building a simple strength checker with conditions
- Combining multiple character sets dynamically
